## 如何完全啟用 PVS ?

### Step 1, Install required tools

- [nodejs](https://nodejs.org/en/download/current/)
- [WebStorm](https://www.jetbrains.com/webstorm/) (建議使用) or [vscode](https://code.visualstudio.com/)

### Step 2, Get dependencies

在前端專案根目錄使用以下指令

- 安裝更好的依賴項目管理器
```bash=
npm i -g pnpm
```

- 安裝依賴項目
```bash=
pnpm i
```

### Step 3, Run

```bash
pnpm start
```
